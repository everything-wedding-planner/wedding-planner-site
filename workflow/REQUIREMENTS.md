# STU-18: Venue Management Page - Admin Dashboard

## Problem
Companies (vendors/venues) need a centralized page within the admin dashboard to manage their venue listings, update venue details, and see performance metrics (inquiries and bookings) for each venue.

## Desired Outcome
A page where companies can view all their venues, edit venue details, add new venues, and see inquiry/booking counts per venue at a glance.

## Scope
**In scope:**
- Display list of venues associated with the company
- View venue details (name, location, description, photos, pricing, availability)
- Edit venue details
- Add new venue
- Show inquiry count per venue
- Show booking count per venue

**Out of scope:**
- Public-facing venue pages (separate feature)
- Venue analytics beyond counts
- Bulk operations on venues
