# STU-15: Company Dashboard

## Problem

Vendors and venues need a centralized place to view and manage all their company information, including profile details, associated services (vendors/venues), and performance metrics. Currently, there's no dedicated page for managing company-level information or viewing company analytics.

## Desired Outcome

A comprehensive company dashboard that allows vendors/venues to:

1. **Profile Management**
   - View and edit company information (name, email, phone, address)
   - Update company details with immediate feedback

2. **Services Overview**
   - See all vendors and venues associated with their company
   - Quick access to manage each service
   - Visual cards showing service type and basic info

3. **Analytics Dashboard**
   - View total bookings count
   - See booking status breakdown (pending, accepted, rejected, cancelled)
   - View total inquiries count
   - See inquiry status breakdown
   - Track recent activity (latest bookings and inquiries)

4. **User Experience**
   - Clean, intuitive layout
   - Responsive design for mobile and desktop
   - Real-time data updates after changes
   - Clear visual hierarchy with sections

## Scope

**In Scope:**
- Company profile viewing and editing
- Associated services (vendors/venues) display
- Basic analytics (counts, status breakdowns)
- Recent activity feed
- Responsive layout

**Out of Scope:**
- Company logo upload (future enhancement)
- Advanced analytics with charts/graphs (future enhancement)
- Company settings/preferences (future enhancement)
- Multi-company support (single company per user)

## Implementation Notes

- **Frontend only** - This is purely frontend work
- Use **placeholder data** where backend endpoints don't exist yet
- Mock data should be realistic and representative of actual API responses
- Structure components so they can easily connect to real endpoints later
